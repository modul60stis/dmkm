Split Data¶

Fungsi train_test_split, secara default akan membagi data menjadi 75% data training dan 25% data test. Untuk mengaturnya dapat menggunakan argument test_size atau train_size. Contoh train_test_split(X, y, train_test = 0.8)

Make Model¶

Untuk mengatur jumlah layer dapat diatur menggunakan parameter hidden_layer_sizes. Sebagai contoh hidden_layer_sizes = [8, 20] berarti model yang dibuat akan menggunakan 2 hidden layer, dimana layer 1 memiliki 8 black box dan layer 2 memiliki 20 black box. Activation function dapat di atur menggunakan parameter activation. Parameter tersebut dapat bernilai relu, logistic, tanh, dan identity Untuk melihat parameter apa saja yang dapat diatur dapat menggunakan fungsi help(MLPClassifier)

In [47]:
layer_size = [8, 20]
activation_function = "relu"

clf = MLPClassifier(hidden_layer_sizes = layer_size, random_state = 0, activation = activation_function, solver = "lbfgs")
clf.fit(X_train_scaled, y_train)
